Chrome blocks microphone on file:// pages. You need to host this on HTTPS.

Easiest free option — GitHub Pages (2 mins):

1. Go to github.com → New repository → name it anything

2. Upload this micstream.html → rename it to index.html

3. Settings → Pages → Branch: main → Save

4. Open https://yourusername.github.io/reponame on your phone

Or use netlify.com/drop — drag the file in, get an HTTPS link instantly.

Bluetooth & Setup ▼
Step 1: Pair BT speaker in phone Settings → Bluetooth
Step 2: Tap ↺ Scan, select your speaker from the dropdown
Step 3: Tap the big Start button

⚠️ BT latency: 80–200ms hardware delay is unavoidable with A2DP Bluetooth. This app minimises software latency. For zero latency use a USB-C→3.5mm cable.
