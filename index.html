Bluetooth Guide ▼
Step 1: Pair BT speaker in Android Settings → Bluetooth
Step 2: Make sure it shows "Connected for Media Audio"
Step 3: Run Diagnostic → Start streaming
Step 4: If no sound, pull down notification bar → tap audio icon → select BT speaker

On reconnect: App auto-restarts after 2s. If still silent, tap Stop then Start.

⚠️ Note: BT adds 80–200ms hardware delay — this is unavoidable in Bluetooth A2DP.
